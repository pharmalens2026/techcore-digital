# Plan: TechCore Solutions Kenya Website

Build a comprehensive, modern ICT solutions website for TechCore Solutions Kenya.

## 1. Visual Design & Theme
- **Style**: Modern Corporate (Minimalist with Tech accents)
- **Palette**: Tech Blue (Primary), Vibrant Teal/Cyan (Secondary), Dark Slate (Neutral), White (Background)
- **Icons**: Lucide React
- **Animations**: Framer Motion (page transitions, scroll reveals)
- **Framework**: React 19, Vite, Tailwind CSS, Shadcn UI

## 2. Information Architecture & Pages
- **Layout**: Sticky Navigation, Mega-menu for Products/Services, Footer with detailed links.
- **Home**: Hero (Value prop), Core Services, Brand Partners, Why Us, CTA.
- **About Us**: Story, Vision/Mission, Team (placeholders).
- **Products**: Category-based listing (Laptops, Servers, Networking, Security) with search/filter.
- **Services**: Detail pages for Networking, CCTV, Cloud, Cybersecurity, IT Support.
- **Industries**: Sector-specific solutions (Education, Healthcare, Corporate).
- **Portfolio**: Case studies grid.
- **Contact**: Multi-field form, Google Maps (placeholder), WhatsApp shortcut.

## 3. Key Components
- `Navbar`: Responsive with mobile drawer.
- `Hero`: High-impact visuals for tech solutions.
- `CategoryCard`: Interactive cards for different product categories.
- `ServiceSection`: Detailed icons and descriptions.
- `PartnerCarousel`: Logo slider for HP, Dell, Cisco, etc.
- `ProductListing`: Grid with "Request Quote" buttons.
- `ContactForm`: Validated form using Shadcn.
- `WhatsAppButton`: Floating widget for instant support.

## 4. Implementation Steps
1. **Setup**: Verify project structure.
2. **Assets**: Generate high-quality tech images (servers, networking gear, modern office, security cameras).
3. **Core Components**: Layout, Navbar, Footer.
4. **Home Page**: Build all sections.
5. **Secondary Pages**: About, Services, Industries, Contact.
6. **Products Page**: Implement filtering and catalog view.
7. **Refinement**: Add animations, responsive tweaks, and accessibility check.
8. **Final Build**: `validate_build`.
